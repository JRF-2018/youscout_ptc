■ About this distribution

This distribution at least includes these PRG for SMILE BASIC of the
Petit Computer.:

  stacklib.prg -- This provides basic functions manipulating the
    variable stack like assembler language.

  stdlib.prg -- This provides "standard" functions as follows.:
    CON_PRINT, PRINT_SIZE_R, M_ALLOC, SET_NTH_SR.

  ctrllib.prg -- This provides functions for "controller" like:
    MNU_CTRL_ADD, MNU_CTRL_LOOP, TCH_CTRL_ADD, TCH_CTRL_LOOP.

  gputlib.prg -- This provides functions to draw character graphics.:
    G_PUT_PM, G_PUT_BM, PM_TO_CHR_R, ROTATE_CHR.

  hlpview.prg -- This provides the HELP Viewer with plenty of escape
    sequences like setting flags and conditional printing.

  example_stacklib.prg -- An example.

  example_hlpview.prg -- An example.


Moreover, this distribution at least includes these Perl scripts.:

  gen_qrcode.pl -- With Data::Petitcom, this can generate QR codes and
    can check some syntax errors and local variable definitions using
    stacklib.prg.

  make_qrcodes_html.pl -- With included template_qrcodes.html, this
    generates QR-code viewer for the Web.

  check_version.pl -- version checker for distribution.
